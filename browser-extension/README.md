# CRM Pilar - Extensão de Monitoramento de Tela

Esta extensão permite o monitoramento de tela para o CRM Pilar, compatível com navegadores baseados em Chromium.

## Navegadores Compatíveis

✅ **Google Chrome** - Totalmente compatível  
✅ **Microsoft Edge** - Totalmente compatível  
✅ **Brave** - Totalmente compatível  
✅ **Opera** - Totalmente compatível  
❌ **Safari** - Não compatível (requer formato diferente)  
❌ **Firefox** - Não compatível (usa outro sistema de extensões)  
❌ **Internet Explorer** - Não suporta extensões modernas  

---

## Instalação no Google Chrome

### 1. Baixar a Extensão
1. Faça login no CRM Pilar
2. Acesse sua página de **Perfil**
3. Na seção "Extensão de Monitoramento", clique em **"Baixar Extensão"**
4. Um arquivo ZIP será baixado

### 2. Extrair os Arquivos
1. Localize o arquivo ZIP baixado (geralmente na pasta Downloads)
2. Clique com o botão direito no arquivo
3. Selecione **"Extrair tudo..."** ou **"Extrair aqui"**
4. Anote o local da pasta extraída

### 3. Instalar no Chrome
1. Abra o Chrome
2. Digite na barra de endereços: `chrome://extensions/`
3. Pressione **Enter**
4. No canto superior direito, ative o **"Modo do desenvolvedor"**
5. Clique em **"Carregar sem compactação"**
6. Navegue até a pasta extraída e selecione-a
7. A extensão aparecerá na lista e na barra de ferramentas

### 4. Fixar a Extensão (Opcional)
1. Clique no ícone de quebra-cabeça (🧩) na barra do Chrome
2. Encontre "CRM Pilar - Monitor de Tela"
3. Clique no ícone de alfinete (📌) para fixar

## Instalação no Android (Kiwi Browser)

O Kiwi Browser é um navegador Android que suporta extensões do Chrome.

### 1. Instalar o Kiwi Browser
1. Abra a Google Play Store
2. Pesquise por **"Kiwi Browser"**
3. Instale o aplicativo

### 2. Baixar a Extensão
1. No Kiwi Browser, acesse o CRM Pilar
2. Faça login e vá para o **Perfil**
3. Baixe a extensão (o ZIP será salvo no dispositivo)

### 3. Instalar a Extensão
1. No Kiwi Browser, toque no menu (⋮)
2. Vá em **"Extensões"**
3. Ative o **"Modo do desenvolvedor"**
4. Toque em **"+ (from .zip/.crx/.user.js)"**
5. Navegue até o arquivo ZIP baixado e selecione-o

## Como Usar a Extensão

### Primeira Configuração
1. Clique no ícone da extensão na barra do navegador
2. Seu **ID de Usuário** será preenchido automaticamente (se estiver logado no CRM)
3. Caso não apareça, copie o ID da sua página de Perfil no CRM

### Iniciar Monitoramento
1. Clique em **"Iniciar Monitoramento"**
2. O navegador exibirá uma janela para selecionar a tela
3. Escolha **"Tela inteira"** (recomendado) ou uma janela específica
4. Clique em **"Compartilhar"**
5. O status mudará para **"Ativo"** ✅

### Parar Monitoramento
1. Clique no ícone da extensão
2. Clique em **"Parar Monitoramento"**
3. O status mudará para **"Inativo"** ⚪

### Inicialização Automática
A extensão pode iniciar automaticamente com o navegador:
1. Marque a opção **"Iniciar com o navegador"**
2. Na próxima vez que abrir o navegador, a extensão estará pronta
3. **Importante**: Por segurança do navegador, você precisará clicar uma vez para ativar e selecionar a tela

## Permissões Especiais (Mac)

No macOS, você precisa conceder permissão de gravação de tela:

1. Vá em **Preferências do Sistema** > **Privacidade e Segurança**
2. Clique em **Gravação de Tela**
3. Marque a caixa ao lado do seu navegador (Chrome/Edge/Brave)
4. Reinicie o navegador se necessário

## Solução de Problemas

### A extensão não conecta
- Verifique se o ID do usuário está correto
- Confirme que você tem conexão com a internet
- Verifique se está logado no CRM Pilar

### O botão fica "Pendente" após reiniciar o navegador
- Isso é normal por segurança do navegador
- Clique em **"Ativar Monitoramento"** para retomar
- Selecione a tela novamente

### Frames não aparecem no dashboard
- Confirme que a extensão mostra status "Ativo"
- Verifique se o supervisor abriu o visualizador
- A transmissão só ocorre quando alguém está visualizando

### Erro no Edge: "Esta extensão não é do Edge Add-ons"
- Isso é apenas um aviso, não um erro
- A extensão funcionará normalmente
- O Edge exibe isso para extensões não baixadas da loja oficial

### Mac: Tela preta ou sem captura
- Verifique as permissões de Gravação de Tela nas Preferências do Sistema
- Reinicie o navegador após conceder a permissão

## Informações Técnicas

- **Versão**: 2.4.0
- **Manifest**: V3 (padrão moderno do Chrome)
- **Captura**: 1 frame a cada 3 segundos
- **Qualidade**: JPEG 50% (otimizado para baixo consumo de dados)
- **Transmissão**: Apenas quando há visualizador ativo (sob demanda)
