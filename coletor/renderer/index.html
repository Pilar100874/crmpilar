🎥 Testar RTSP (mesma engine do "Ao Vivo")

Simula exatamente o que o coletor faz para transmitir "Ao Vivo": abre o RTSP via ffmpeg por 1s e reporta codec, resolução ou o erro específico (401, 404, timeout, HEVC…). Também substitui o teste no VLC.

Como testar no VLC (alternativa manual)
1. Abra o VLC → Mídia → Abrir Fluxo de Rede (Ctrl+N).
2. Cole a URL RTSP no formato rtsp://usuario:senha@ip:554/path.
3. Clique Reproduzir. Se o vídeo aparecer, o RTSP está OK — o problema é do coletor/PC (firewall, HEVC, versão).
4. Se der erro no VLC, o problema é da câmera (RTSP desabilitado, path errado, credencial ou HEVC).
